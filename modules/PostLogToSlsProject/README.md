# PostLogToSlsProject

## Usage

Create a policy that allow to post log to specified SLS project.

```hcl
module "example" {
  source = "terraform-alicloud-modules/ram-policy/alicloud//modules/PostLogToSlsProject"
  sls_project = "prj1"
  sls_logstores = ["logstore1", "logstore2"]
}
```
